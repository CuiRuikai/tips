# Linux Command Line Cheat Sheet

## File operation

```
ls | wc -l  # count file number in a directory
du -sh  # count dist usage by Gigabyte
find . -name "*.ply" -type f -delete
```


## running command

```
nohup command > output.log &  # running command without hop
```
